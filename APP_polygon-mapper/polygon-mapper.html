Polygon Drawing Tool

Instructions:
1. Use the polygon tool (square icon) on the left to draw
2. Click points to create polygon, double-click to finish
3. WKT output updates automatically in EPSG:3059

Important: If map tiles don't load, serve this file from a web server instead of opening it directly. Run: python -m http.server 8000 then open http://localhost:8000/polygon-mapper.html
